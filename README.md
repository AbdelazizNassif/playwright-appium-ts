setup instructions for appium tests:
- download node js
- install appium: npm install appium then download UIautomator
- download android studio then install Pixel XL android 13 virtual device
- start appium server
- open the virtual device
- download the project, run npm install to download dependencies
- open terminal, then run command npm run test:mobile

setup instructions for playwright tests:
- download node js
- download the projct
- run npm install
- run npx playwright test
